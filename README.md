# Apoyo Campaña Pacto Histórico en la Ciudad Mitrada

## Descripción General

Apoyo Campaña Pacto Histórico en la Ciudad Mitrada es una aplicación web desarrollada para apoyar las actividades de planeación, organización y análisis de información territorial durante campañas políticas y procesos de participación ciudadana.

El sistema permite gestionar eventos, registrar información territorial, analizar datos provenientes de encuestas y procesar opiniones ciudadanas mediante técnicas de análisis semántico, facilitando la identificación de problemáticas, tendencias y argumentos predominantes dentro de distintos sectores de la población.

La solución integra herramientas de análisis estadístico y consulta documental para transformar información recolectada en campo en conocimiento útil para la toma de decisiones, la planificación de actividades territoriales y la preparación de estrategias de intervención sustentadas en evidencia.

## Objetivos del Sistema

- Gestionar actividades y eventos territoriales.
- Consolidar información proveniente de encuestas ciudadanas.
- Analizar indicadores estadísticos relacionados con participación política y problemáticas sociales.
- Identificar patrones y argumentos recurrentes en opiniones ciudadanas.
- Relacionar problemáticas identificadas con documentos, propuestas y planes de acción registrados por la organización.
- Facilitar la consulta de información mediante mecanismos de búsqueda semántica.

## Módulos Principales

- Módulo de Gestión de Eventos

Permite la planificación, programación y seguimiento de actividades territoriales, incluyendo la gestión de participantes, responsables y recursos asociados a cada evento.

- Módulo de Análisis Estadístico Territorial

Permite importar y procesar información proveniente de encuestas, generando indicadores, distribuciones y cruces estadísticos que apoyan la caracterización territorial y la toma de decisiones.

- Módulo de Análisis Semántico y Consulta Documental

Permite clasificar opiniones ciudadanas, identificar argumentos recurrentes, asociar problemáticas con documentación institucional y realizar búsquedas semánticas sobre documentos previamente registrados.
